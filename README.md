📌 Local-Store-E-Commerce-Platform

The Local Store E-Commerce Platform is a web application that allows users to browse products, add items to a shopping cart, and place orders online. The platform simulates a basic online store for a local business, providing an interactive and user-friendly shopping experience.

✨ Key Features

- Product listing with images, descriptions, and prices
- Add and remove items from the shopping cart
- Product search, sorting, and filtering
- Checkout system for placing orders
- Order history tracking
- User login simulation
- Responsive design for mobile and desktop
- Data persistence using localStorage

🛠 Tech Stack

- Frontend: HTML5, CSS3, JavaScript (Vanilla JS)
- Storage: Browser localStorage (no backend, no database)
- Tools: VS Code, Live Server

💻 Languages Used

- HTML – Structure of all web pages
- CSS – Styling, layout, responsiveness, and animations
- JavaScript – Application logic including cart, orders, login, search, filter, and sort functionality
